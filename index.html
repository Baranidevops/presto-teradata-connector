About the initiative.

What is Presto?

Presto is an open source distributed SQL query engine for running interactive analytic queries against data sources of all sizes ranging from gigabytes to petabytes. Among its features, it enables you to connect to other databases using some connector, in order to perform queries and joins over several sources.

What problem do we solve?

Some databases require special license agreements in order to use their drivers, so there are many connectors that cannot be merged into Presto. Despite cannot be merged into Presto, we can provide a single place where to register those projects, so that they reach visibility and enhance collaboration within Presto community.

Our objectives are to provide:

  • a web page where developed connectors are listed and can be downloaded
  • a template project with instructions, to reduce time for development of new projects
  • a groupId and pom.xml snippets, so that connectors can be released and grouped at Maven Central with minimum effort

Presto connectors

What is Presto?

Presto is an open source distributed SQL query engine for running interactive analytic queries against data sources of all sizes ranging from gigabytes to petabytes. Among its features, it enables you to connect to other databases using some connector, in order to perform queries and joins over several sources.

Versioning

We use a major.minor.patch version, where major.minor match Presto version the connector is compatible with; and patch refers to versions of the connector for that Presto version.

About the initiative.

What problem do we solve?

Some databases require special license agreements in order to use their drivers, and those connectors cannot be merged into Presto. By providing a single place where to list those projects, they reach visibility and enhance collaboration within Presto community.

Our goals are to provide a:

  • web page where developed connectors are listed and can be downloaded
  • template project with instructions, to reduce time of new connectors development
  • groupId and pom.xml snippets, so that connectors can be released and grouped at Maven Central with minimum effort
  • unified versioning convention for connectors
